# some

数组有个 `some()` 方法. 返回 `bool` 值.

情况和 `every()` 相反.

[every()](./array-every.md) 是要求回调函数每次都返回 true, `every()` 执行的结果才是 true.

而 `some()` 只要回调函数有一次返回 true, 那么 `some()` 就会返回 false.
